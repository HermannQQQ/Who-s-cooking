Readme file for project: where is the chef from?

pre-requisites
--------------
Make sure you have installed: 
* python 3.6.x
* pipenv (pip install pipenv).

setup
-----
* In a dir of your choice: 
git clone https://github.com/manishpandit/chef.git
* cd to chef
cd chef
* sync pipenv: this will install required packages.
pipenv sync

use
---
* cd to dir: chef
* switch to pipenv
pipenv shell
* open jupyter notebook. This will open a browser window.
jupyter notebook
* In the browser, open the chef notebook.
* Start running each code cell. If you run into to SSL cert error during 
* importing the json file. Open finder--applications--python3.6 and run 
* install cert. retry.
